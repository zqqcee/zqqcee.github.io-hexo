JSON

JavaScript Object Notation

  • 本质上（从数据格式上）是JS的对象

  • 保存后是文本

  • 文本与JS的对象是可以对等转换的

    这点与csv不同，js 读取 csv 的时候会转化成数组
    但是js 读取JSON文件时 ，是什么数据就会读出什么数据
